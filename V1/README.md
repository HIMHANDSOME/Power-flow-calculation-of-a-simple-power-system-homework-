# 电力系统潮流计算与分析

##  项目概述

本项目用于完成某地区电力系统的潮流计算与分析任务，基于给定的系统参数进行完整的电力系统分析，包括标幺值计算、节点导纳矩阵生成、节点类型分类和潮流计算。

### 原始任务要求（来自"某地区电力系统信息.txt"）

1. **计算各元件的标幺值参数**
2. **生成电力网络的节点导纳矩阵 (Ybus)**
3. **指出各节点应设置为何种类型**（平衡节点、PV节点、PQ节点）
4. **依据给定的运行条件计算电网潮流**

### 系统特点
- 包含2个火电厂、1个风电场、3个变电所
- 220kV主干网，9节点系统
- 考虑了线路参数、变压器参数、发电机和负荷特性
- 支持中文字符显示和中文报告生成

### 主要文件说明

#### `flow.py` - 主程序
- **功能**：完整的潮流计算与分析
- **输入**：内置系统参数（基于"某地区电力系统信息.txt"）
- **输出**：
  1. 控制台输出：详细计算过程和结果
  2. JSON文件：完整的潮流计算结果
  3. 5张分析图表：网络拓扑、电压分布、线路负载等

- **功能**：将JSON结果中的Ybus矩阵转换为Markdown表格
- **用法**：
  ```bash

  python json_ybus_to_md.py

  # 指定JSON文件
  python json_ybus_to_md.py power_flow_results_20260408_121404.json
  ```
- **输出**：生成两个Markdown文件（完整版和简化版）

##  功能特性

1. **精确的标幺值计算**
   - 基准容量：100 MVA
   - 基准电压：220 kV
   - 自动计算线路、变压器标幺值参数

2. **节点导纳矩阵生成**
   - 自动提取9×9复数矩阵
   - 支持Markdown格式导出

3. **节点类型智能识别**
   - 平衡节点 (Slack)：发电厂1（母线1）
   - PV节点：发电厂2（母线2）
   - PQ节点：风电场、变电所、联络节点

4. **潮流计算**
   - 使用Newton-Raphson法
   - 自动检查收敛性
   - 输出详细潮流分布

5. **数据导出与可视化**
   - JSON格式完整结果保存
   - 5种专业图表生成
   - 支持中文标签和说明

6. **Ybus矩阵文档化**
   - 自动生成Markdown表格
   - 集成节点名称信息
   - 复数格式化显示

##  注意事项

### 常见问题

1. **中文显示问题**
   - 确保系统安装了中文字体
   - 脚本已包含字体自动检测功能
   - 如果仍有乱码，可手动指定字体路径

2. **依赖包安装失败**
   ```bash
   # 尝试使用镜像源
   pip install pandapower -i https://pypi.tuna.tsinghua.edu.cn/simple

   # 或使用conda安装
   conda install -c conda-forge pandapower
   ```

3. **绘图功能缺失**
   - 如需高级网络拓扑图，安装igraph：
   ```bash
   pip install python-igraph
   # 或
   pip install pandapower["all"]
   ```

4. **JSON文件读取失败**
   - 确保使用UTF-8编码
   - 检查文件路径是否正确
   - 使用最新的JSON文件

### 性能优化
- 对于大规模系统，可调整Newton-Raphson法的收敛容差
- 考虑使用更高效的稀疏矩阵存储
- 并行计算选项
